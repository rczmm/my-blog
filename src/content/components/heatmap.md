### 组件特性
- **动态色阶**：根据数据权重自动切换颜色深浅。
- **交互提示**：悬停时显示具体日期和贡献次数。
- **灵活布局**：支持周、月、年等不同时间维度的展示。

### 技术实现
基于 `SVG` 或 `Canvas` 渲染，确保在大量数据下的性能表现。

### 示例配置
```javascript
const data = [
  { date: '2023-01-01', count: 5 },
  { date: '2023-01-02', count: 12 },
  // ...
];
```
