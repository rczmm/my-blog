## Markdown 简介

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

## 基础语法

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调
```markdown
**粗体文本**
*斜体文本*
~~删除线~~
```

### 列表
```markdown
- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2
```

### 代码
```markdown
`行内代码`

​```javascript
// 代码块
console.log("Hello, World!");
​```
```

## 高级技巧

- 使用表格组织数据
- 添加链接和图片
- 创建任务列表
- 使用引用块
