## Astro 是什么？

Astro 是一个现代化的静态站点生成器，专为构建快速、内容丰富的网站而设计。

### 核心特性

1. **零 JavaScript 默认**：默认情况下不向客户端发送 JavaScript
2. **组件岛架构**：只在需要时加载交互式组件
3. **多框架支持**：支持 React、Vue、Svelte 等

## 开始使用

```bash
npm create astro@latest my-project
cd my-project
npm run dev
```

这就是 Astro 的强大之处！
